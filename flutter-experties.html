Advanced / Tricky Questions & Answers

  • 1. How do you optimize Flutter app performance for complex screens?
    Use const widgets, avoid unnecessary rebuilds, ListView.builder/GridView.builder, cache images, profile with DevTools.
  • 2. How do you reduce app bundle size?
    Remove unused assets/packages, enable ProGuard, deferred loading, compress images, tree shaking for Dart code.
  • 3. How do you handle memory leaks?
    Dispose controllers, listeners, streams, focus nodes, monitor memory usage with DevTools.
  • 4. How do you handle complex animations efficiently?
    AnimationController with TickerProviderStateMixin, AnimatedBuilder, offload heavy computations outside callbacks.
  • 5. How do you implement platform-specific code?
    Use platform channels for native code communication.
  • 6. How do you handle large lists?
    ListView.builder/SliverList, pagination, memoize items with const constructors.
  • 7. How do you manage asynchronous streams and real-time data?
    StreamBuilder, RxDart, handle errors and dispose properly to avoid leaks.
  • 8. How do you secure sensitive data?
    Flutter Secure Storage, HTTPS, proper token storage and encryption.
  • 9. How do you manage multiple environments?
    Flavor configurations, separate entrypoints, flutter_dotenv/config classes for environment-specific variables.
  • 10. How do you debug production crashes not in development?
    Crashlytics/Sentry, reproduce in release builds, check minification, logging and stack traces.
  • 11. How do you handle widget rebuilds and state efficiency?
    Const constructors, split widgets, Provider/Riverpod/Bloc selectively.
  • 12. How do you optimize network requests?
    Caching, debounce/throttle, compress payloads, retry errors efficiently.
  • 13. How do you handle image-heavy apps efficiently?
    cached_network_image, prefetch images, compress assets, use WebP format.
  • 14. How do you implement background services?
    WorkManager for Android, BackgroundFetch/FlutterWorkManager, proper lifecycle handling.
  • 15. How do you implement unit, widget, and integration tests?
    Unit tests with test, widget tests with flutter_test, integration tests with integration_test/Flutter Driver.
